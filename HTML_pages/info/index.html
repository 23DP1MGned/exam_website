A little bit
Information about
our project

CryptoWallet project is a cryptocurrency
wallet simulator written in Python. It provides
a secure and user-friendly platform for exploring the
functionalities of a cryptocurrency wallet. Key features include
user registration with secure JSON-based data storage, wallet creation, deletion,
viewing, and switching between multiple wallets. Each wallet has a unique address and can
hold various cryptocurrencies. Users can top up their balance in USDT, convert it to other coins,
and withdraw to a card. Transactions are automatically logged and can be viewed, sorted, and filtered.
All critical data is securely saved, with confirmation prompts before sensitive actions like exiting the app. Designed for both beginners and experienced users, it offers an easy way to manage digital assets safely.
