Step 1: Registration

1. Registration:
 - Launch the application and start registration.
- Enter your username and password.
- The data will be saved to the `user_data.json` file, and you will see a message confirming successful registration.
2. Registration Confirmation:
 - After registration, the application will automatically check for user data.
- If user data is not found, you will be prompted to register again.
Step 2: Wallet Management

1. Creating a Wallet:
- After logging into the system, select the option to create a new wallet.
- Enter a name for your new wallet.
- The wallet will be created with a unique address and saved to a file in the `wallets` directory.
2. Deleting a Wallet:
- Select the option to delete an existing wallet.
- Enter the name of the wallet you want to delete.
- The wallet will be removed from the system.
3. Viewing Wallets:
- Select the option to view all your wallets.
- You will see a list of all available wallets with their balances.
4. Switching Between Wallets:
- Select the option to choose the active wallet.
- Enter the wallet number from the list of available wallets.
- The active wallet will be selected for performing operations.
5. Transferring Funds Between Wallets:
- Select the option to transfer funds between wallets.
- Choose the sender and recipient from the list of available wallets.
- Enter the amount and cryptocurrency for the transfer.
- The funds will be transferred between the wallets.
Step 3: Wallet Operations

 1. Topping Up Balance in USDT:
 - Select the "Top up balance in USDT" option.
- Enter the amount in USDT and your card details.
- Your wallet balance will be topped up.
2. Converting USDT to Cryptocurrency:
 - Select the "Convert USDT to Crypto" option.
- Enter the amount in USDT and choose the cryptocurrency for conversion.
- The funds will be converted and added to the balance of the selected cryptocurrency.
3. Converting Cryptocurrency to USDT:
 - Select the "Convert Crypto to USDT" option.
- Enter the amount of cryptocurrency and choose USDT for conversion.
- The funds will be converted and added to the USDT balance.
4. Withdrawing Funds:
 - Select the "Withdraw funds" option.
- Enter the amount and cryptocurrency for withdrawal, as well as your card details.
- The funds will be transferred to your card.
5. Viewing Cryptocurrency Balances:
 - Select the "View your crypto" option.
- You will see a list of all available cryptocurrencies with their balances.
Step 4: Transaction Management

1. Creating a Transaction:
- Transactions are created automatically when topping up the balance, converting cryptocurrencies, and withdrawing funds.
- All transactions are saved to the `user_transactions.json` file.
2. Viewing Transactions:
- Select the option to view the transaction history.
- You will see a list of all your transactions with the ability to sort by date, amount, type, and cryptocurrency pairs.
3. Sorting Transactions:
- When viewing the transaction history, you can choose the option to sort transactions by various parameters:
- Date: Sort by transaction date in ascending or descending order.
- Amount: Sort by transaction amount in ascending or descending order.
- Type: Sort by transaction type (e.g., top-up, conversion, withdrawal).
- Cryptocurrency Pairs: Sort by cryptocurrency pairs (e.g., USDT -> BTC).
4. Filtering Transactions:
- You can filter transactions by type or cryptocurrency pairs:
- Transaction Type: Enter the transaction type for filtering (e.g., top-up, conversion).
- Cryptocurrency Pairs: Enter the source and target cryptocurrencies for filtering transactions (e.g., USDT -> BTC).
Step 5: Exiting the System

1. Saving Data:
- Before exiting the system, the application will automatically save all data of the active wallet.
2. Exiting:
- Select the 'Exit' option.
- The application will terminate, and you will see a message confirming the exit.
